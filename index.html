Application Overview (from an end user perspective):

This application will provide users with a webpage that is able to test capabilities and skills of a Computer Science or related field in their coding abilities and conseptual knowledge. It implements common coding problems with desired outputs and prompts the user to code the solution to the problem. It also provides the users a section to build a mock interview where it will ask the user a series of coding and conseptual questions and give a grade based on the users answers. It provides external resources in order to help user to improve knowlegde and skills.
